YAC Cannabinoid Compound Library

Rational Design of Pathway-Selective Cannabinoid Compounds for ADHD: Dissociating Therapeutic Efficacy from Orexigenic Effects

Overview

This research presents a comprehensive rational drug design framework for creating THC analogs that preserve ADHD-relevant prefrontal cortex modulation while eliminating appetite stimulation through three mechanistically distinct strategies.

View on GitHub Read Paper (PDF)

Three Strategies

Strategy A: Biased Signaling

Gi-selective partial agonists and allosteric-orthosteric hybrids that preferentially activate therapeutic pathways through defined structural modifications.

Compounds: YAC-101, YAC-102

Strategy B: Prodrug Activation

CYP2D6- and esterase-activated prodrugs achieving cortex-selective activation while maintaining blood-brain barrier penetration.

Compounds: YAC-201, YAC-202

Strategy C: Multi-Target Ligands

Hybrid molecules engaging CB1 while antagonizing appetite-related receptors (GHSR1a, NPY Y1) or agonizing anorexigenic receptors (MC4R).

Compounds: YAC-301, YAC-302

Quick Start

# Clone and install
git clone https://github.com/mlong/yac-cannabinoid-adhd.git
cd yac-cannabinoid-adhd
pip install -r requirements.txt

# Analyze compounds
python src/compounds.py

# Generate molecular visualizations
python src/visualize.py

Python API

from src.compounds import YACCompoundLibrary, MolecularAnalyzer

library = YACCompoundLibrary()
compound = library.get_compound("YAC-201")

analyzer = MolecularAnalyzer()
props = analyzer.analyze_compound(compound.smiles)
print(f"TPSA: {props['tpsa']} A^2")
